# Requirements

## 1. Overview

The Night Desk 是一款以 ghostpixel 为主角的竖屏决策生存游戏：玩家在超自然酒店完成 12 次午夜值班决定，维持镇定、秩序、信任和真相直到清晨。

## 2. Visual Design

- 设计画布为 390 × 844 px，按视口宽高等比缩放，不产生页面滚动。
- 主色为墨黑 `#090B14`、靛蓝 `#15182A`、纸白 `#F2E9D8`、薄荷 `#9EF4D3`、黄铜 `#D6AD63` 和警报珊瑚 `#FF6F7D`。
- 顶部 102 px 为四项仪表，中部 322 px 为角色舞台，底部 386 px 为事件账簿。
- 事件标题为 24～30 px 衬线体，正文为 13 px / 19 px，选择按钮高度至少 64 px。
- 角色资产包括 1 张锚点、7 张透明状态立绘和 4 张 768 × 1376 剧情图；封面为 1024 × 1024 transit 叙事海报。
- 选择按下 110 ms 缩至 0.97；立绘在 220 ms 内切换；结果面板在 420 ms 内滑入；结局光晕持续 900 ms。

## 3. Game Mechanics

- 一局固定 12 回合，四项状态初始值均为 55，限制在 0～100。
- 18 张事件牌每局不重复抽取 12 张；事件按回合范围和隐藏标记筛选。
- 每个选择改变 2～4 项状态，单项变化为 −18～+16；输入锁定 1100 ms。
- 第 6 回合结束后展示调查或服从两种中点之一。
- 任一状态低于 18 时强制使用 exhausted 立绘并每 1400 ms 播放危机脉冲。
- 得分为 `回合×100 + 状态总和×4 + 最低状态×6 + 发现数×150 + 通关800`。
- 历史最高分写入 `portrait_quest_best_night-desk`，首次读取兼容旧键 `portrait_quest_best`。

## 4. Controls

- Pointer / Touch：开始、左右选择、中点继续、再来一次和返回首页均使用单次 `pointerdown`。
- Keyboard：A / 左方向键选择左项，D / 右方向键选择右项，Enter 用于开始、继续和重开。
- 开局前 1500 ms 与每次选择后的 1100 ms 内忽略额外输入。

## 5. Win / Lose Conditions

- 完成第 12 回合且四项状态均大于 0 即胜利，根据最高状态和隐藏标记进入 5 种结局之一。
- 任一状态归零立即失败，按镇定、秩序、信任、真相顺序选择 4 种失败结局。
- 结算页显示全屏结局图、用户名、结局标题、总结、最终分、历史最高、四项数值、重开和返回按钮。

## 6. Sound Effects

- 开始：440 Hz 与 660 Hz 正弦波，各 90 ms，峰值 0.07。
- 点击：620→420 Hz 三角波，55 ms，峰值 0.035。
- 正向反馈：520 / 660 / 820 Hz 正弦波，各 70 ms，峰值 0.045。
- 负向反馈：180→95 Hz 锯齿波，180 ms，峰值 0.045。
- 中点：120→360 Hz 正弦滑音 700 ms；胜利为 392 / 523 / 659 / 784 Hz 上行；失败为 220 / 174 / 110 Hz 下行。
