# The Night Desk

以 `ghostpixel` 为主角的竖屏超自然酒店决策游戏，也是“多状态人物立绘 + 全屏关键帧”主角模板的第一个独立发布样本。

```bash
npm install
npm run dev
npm run build
```

- 游戏条目：`the-night-desk`
- 主角：`ghostpixel`
- 主题：午夜酒店值班
- 美术：1 张角色锚点、7 张状态立绘、4 张剧情关键帧
- Remix：完整源码通过 GitHub 仓库 zip 提供，构建 `base: './'`

项目记录位于 `doc/`。平台 UUID、线上 URL、仓库和发布验证写入 `doc/publish-record.md`。
